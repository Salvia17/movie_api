myFlix Project

Creating a REST API with Node.js



Objective:

To build the server­side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.


Accesing movies collection


HTTP method URL endpoint Business logic Request format Response format
GET /movies Return a list of ALL movies None A JSON object with all movies data
GET /movies/:Title Return data about a single movie by title None A JSON object with single movie data
GET /movies/genres/:Name Return data about a genre (description) by name (e.g., “Crime”) None A JSON object with data about genre
GET /movies/directors/:Name Return data about a director (bio, birth year, death year) by name None A JSON object with data about director

Accesing users collection


HTTP method URL endpoint Business logic Request format Response format
POST /users Allow new users to register A JSON object A JSON object with data of new user
PUT /users/:Username Allow users to update their user info A JSON object A JSON object with updated user data
POST /users/:Username/Movies/:MovieID Allow users to add a movie to their list of favorites None An updated array of user's favorites
DELETE /users/:Username/Movies/:MovieID Allow users to remove a movie from their list of favorites None An updated array of user's favorites
DELETE /users/:Username Allow existing users to deregister None Message confirming that account was deleted
